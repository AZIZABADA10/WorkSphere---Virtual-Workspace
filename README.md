# WorkSphere --- Application de Gestion Visuelle du Personnel

## *Projet réalisé dans le cadre d'un brief croisé (Front-End + UI/UX)*

------------------------------------------------------------------------

## Description du projet

WorkSphere est une application web interactive permettant de gérer le
personnel d'une entreprise directement depuis un plan d'étage
dynamique.\
L'objectif est d'offrir une interface moderne, visuelle et intuitive
pour organiser et affecter les employés en respectant des règles métier
strictes.

------------------------------------------------------------------------

## Objectifs du projet

-   Interface intuitive et responsive\
-   Ajout / gestion des employés\
-   Règles d'accès basées sur les rôles\
-   Sauvegarde automatique (localStorage)\
-   Plan d'étage interactif\
-   Profil complet des employés\
-   Déploiement sur GitHub Pages

------------------------------------------------------------------------

## Fonctionnalités principales

### Gestion des employés

-   Ajout via modale (Nom, Rôle, Email, Téléphone, Photo, Expériences)
-   Prévisualisation des photos
-   Liste des employés non assignés
-   Suppression / réassignation
-   Affichage détaillé (profil)

### Plan d'étage interactif

Zones : 1. Salle de conférence\
2. Réception\
3. Salle des serveurs\
4. Salle de sécurité\
5. Salle du personnel\
6. Salle d'archives

Chaque zone contient un bouton permettant d'ajouter un employé autorisé.

### Règles métier

-   Réception → Réceptionnistes uniquement\
-   Salle des serveurs → Techniciens IT\
-   Salle de sécurité → Agents de sécurité\
-   Manager → partout\
-   Nettoyage → sauf archives\
-   Autres rôles → zones non restreintes
